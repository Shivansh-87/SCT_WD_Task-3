# Tic Tac Toe 

This is a modern, vibrant Tic Tac Toe web application with a Gen Z inspired design. It allows you to play either against another human or against the computer. The app is fully responsive, accessible, and features smooth animations with a playful color scheme.

---

## Features

- Play Tic Tac Toe against a human opponent or a simple AI computer.
- Responsive layout adapting perfectly to mobile, tablet, and desktop screens.
- Keyboard accessible with focus management and ARIA attributes for screen readers.
- Stylish design with vibrant gradients, smooth hover effects, and clean typography.
- Material Design icons for intuitive controls.
- Clear game status messages and animations highlighting the winner.
- Reset button to start a new game anytime.
- No overlapping elements with generous spacing for comfortable interaction.

---

## How to Use

1. Open the `index.html` file in any modern web browser.
2. Choose the game mode:
   - **Human**: Play against another player taking turns.
   - **Computer**: Play against the computer's AI.
3. Click or tap on an empty cell to place your mark (X starts).
4. The game will automatically detect wins or draws and highlight the winning cells.
5. Use the reset button to start a new game anytime.
6. Keyboard users can navigate the cells with Tab and place marks with Enter or Space keys.

---

## Technologies Used

- HTML5, CSS3 with modern features (Flexbox, CSS Grid, custom properties)
- Vanilla JavaScript (ES6+), modular and self-contained
- Google Fonts (Poppins)
- Google Material Design Icons via CDN

---


## Accessibility

- Fully keyboard navigable with visible focus states.
- ARIA roles and labels for grid and cells.
- Live region updates for announcing game status.
- Large interactive areas with sufficient contrast.

---

## Customization

You can easily customize colors, fonts, and animations by editing the CSS section in the HTML file. The game logic is simple and well documented for easy extension or improvement of the AI.
